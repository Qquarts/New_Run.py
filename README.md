## 🧬 개요 (Overview)

**PHAM Neural Genesis**는  
과학적 뉴런 모델링, 블록체인 기반 기록, 그리고 오픈 코드 문화를 결합한  
세계 최초의 **“신뢰 기반 오픈 비즈니스 시스템”**입니다.

> 🪶 **코드는 제품이다. Ledger는 증거다. 신뢰는 화폐다.**

본 프로젝트는 코드 그 자체가 자산이며,  
모든 수정과 발행이 Ledger(원장)에 투명하게 기록되는 새로운 경제 구조를 제시합니다.

## 🌍 작동 방식 (How It Works)

1. **코드 실행 및 연구**
   ```bash
   python3 New_Run.py
→ 뉴런 에너지 순환 시뮬레이션 실행.

2. Ledger 서명 등록
python3 pham_sign_v2.py New_Run.py "사용자이름" "메시지"

→ 코드의 해시값이 블록으로 생성되어 pham_chain_log.json에 자동 저장됩니다.
→ 필요 시 IPFS에 CID가 함께 발행됩니다.

3. Ledger 검증
python3 view_chain.py
→ 전체 블록체인 기록을 확인합니다.
